如何添加音乐

第 1 步：把 .mp3 文件放到 assets/music/ 目录下
第 2 步：打开 components/music-player/music-player.js，找到顶部的 CATEGORIES 对象
第 3 步：在对应分类的 tracks 数组中添加一行

添加示例

// 在 music-player.js 顶部找到 CATEGORIES：

const CATEGORIES = {
  /* --- 说唱 --- */
  rap: {
    name: '说唱',
    icon: '🎤',
    tracks: [
      // 👇 添加你的说唱歌曲
      { title: 'song name', src: 'assets/music/song.mp3' },
      { title: '另一首',   src: 'assets/music/another.mp3' },
    ],
  },

  /* --- 流行 --- */
  pop: {
    name: '流行',
    icon: '🎵',
    tracks: [
      { title: 'payphone',  src: 'assets/music/payphone.mp3' },
      // 👇 继续添加
    ],
  },

  /* --- 摇滚 --- */
  rock: {
    name: '摇滚',
    icon: '🎸',
    tracks: [
      // 👇 添加你的摇滚歌曲
    ],
  },

  /* --- 纯音乐 --- */
  instrumental: {
    name: '纯音乐',
    icon: '🎹',
    tracks: [
      { title: 'sakana', src: 'assets/music/sakana.mp3' },
    ],
  },

  /* --- (ﾟω´) --- */
  special: {
    name: '(ﾟω´)',
    icon: '(ﾟω\')',
    tracks: [
      // 👇 添加特殊收藏
    ],
  },
};

格式说明

{ title: '显示的歌名', src: 'assets/music/文件名.mp3' }
  • title — 播放列表中显示的歌曲名称
  • src — MP3 文件的路径，相对于网站根目录

添加新分类（可选）

// 1. 在 CATEGORIES 中添加新分类
jazz: {
  name: '爵士',
  icon: '🎷',
  tracks: [],
},

// 2. 在 CATEGORY_ORDER 数组中加入 key（控制显示顺序）
const CATEGORY_ORDER = ['rap', 'pop', 'rock', 'instrumental', 'special', 'jazz'];

快捷 API

MusicPlayer.play()             // 播放
MusicPlayer.pause()            // 暂停
MusicPlayer.next()             // 下一首
MusicPlayer.prev()             // 上一首
MusicPlayer.toggle()           // 切换播放/暂停
MusicPlayer.switchCategory('rock')  // 切换到摇滚分类

键盘快捷键

  • 空格 — 播放 / 暂停
  • ← / → — 快退 / 快进 5 秒
  • ↑ / ↓ — 音量 +/- 5%
  • M — 静音 / 取消静音
